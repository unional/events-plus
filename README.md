Provides additional features to event emitters.

Supports the following emitters:

- [events]
- [eventemitter2]
- [eventemitter3]
- [fbemitter]
- [EventTarget]

## Installation

```sh
# npm
npm install events-plus

# yarn
yarn add events-plus

# pnpm
pnpm install @just-web/app

#rush
rush add -p @just-web/app
```
